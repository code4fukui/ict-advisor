# ICTアドバイザー派遣事業

総務省が実施する地域情報化アドバイザー派遣事業に関する要綱です。地域のICT利活用を支援するため、専門家であるアドバイザーを派遣し、課題解決や情報提供などの支援を行います。

## 事業目的
- 地域におけるICT利活用の促進と活力ある地域づくりへの寄与
- 地域におけるICT活用の中核人材の育成

## 事業内容
- 地域のICT活用に関する課題整理、アドバイス・提言、情報提供等の支援
- 地方公共団体や関係団体からの申請に基づき、アドバイザーを派遣
- 原則3日間以内、1日7時間以内の派遣
